# SQL Projects

This folder contains my SQL projects and practice work, organised by type. Each subfolder has its own README with more details.

## Folders

**[case-studies](./case-studies)**
Real-world business case studies solved using SQL. Each case study includes the problem statement, the queries written, and the insights drawn from the data.

**[data-lemur](./data-lemur)**
SQL practice problems from the DataLemur platform. These cover a range of difficulty levels and topics including joins, window functions, aggregations, and subqueries.

**[data-warehouse-project](./data-warehouse-project)**
An end-to-end data warehouse built with SQL Server. This project follows Medallion Architecture (Bronze, Silver, Gold layers) and delivers a star schema model ready for business analytics. It covers ETL pipeline design, data cleaning, data modelling, and SQL-based reporting.

## Skills Covered

- Writing and optimising SQL queries
- Joins, subqueries, and window functions
- Data cleaning and transformation
- ETL pipeline design
- Dimensional modelling and star schema
- Business analytics and reporting
